## 🚀 快速开始

### 启动服务

```bash
cd frontend
python -m http.server 8080
```

### 访问界面

在浏览器中打开: **http://localhost:8080**

---

## 🔍 如果看到文件列表

如果访问 `http://localhost:8080` 后看到文件列表页面，请：

1. 点击 **index.html** 链接
2. 或直接在地址栏输入: `http://localhost:8080/index.html`

## ❓ 常见问题

### Q: 为什么看到旧版本的界面？

**A**: 浏览器缓存问题。解决方法：
- Mac: `Cmd + Shift + R`
- Windows: `Ctrl + Shift + R`

### Q: 界面没有响应？

**A**: 检查后端服务是否运行：
```bash
curl http://localhost:8000/health
```

应该返回: `{"status":"healthy","service":"Semantic Query System"}`

### Q: 看到文件列表而不是界面？

**A**: 点击 index.html 链接，或访问 http://localhost:8080/index.html
